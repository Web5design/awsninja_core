
AWSNinja core files -  Version 0.0

INTRODUCTION
------------

This package contains the common files needed by other packages in the
AWSNinja framework.  It is not usable by itself.


REQUIREMENTS
------------

PHP 5.0+


INSTALLATION
------------

  1. Copy the files to a location of your choosing.
  
  2. Modify the file ./config.samp.php to add your AWS credentials and database
     credentials.  Then, add the contents of of the config.samp.php file of the
     AWSNinja package file you are installing to the core config.samp.php
  
  3. Rename config.samp.php to config.php.
  
  4. View the README for the AWSNinja package you are installing.
